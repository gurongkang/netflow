Privacy & Permissions

Transparent About What We Access

We only request permissions that are strictly necessary. Here's exactly what we use and why.

🎤

Microphone

Used only to detect clap and whistle sounds. Active only when your screen is off. Audio is never recorded or transmitted.

🔔

Notifications

Required to show a persistent status when detection is active, so the service continues running in the background.

🏠

Default Launcher

Needed to replace your home screen. You can switch back to any other launcher at any time from system settings.

📱

Device Sensors

Accelerometer is used only for the optional Anti-Theft motion detection feature. Disabled when anti-theft is off.
